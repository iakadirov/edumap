## 1. Объективные метрики (40 баллов)

Данные, которые можно проверить документально.

| Критерий | Макс. баллов | Как считаем |
|----------|--------------|-------------|
| **Лицензия и аккредитация** | 5 | Есть/нет + тип аккредитации |
| **Квалификация учителей** | 10 | % с высшим пед. + сертификаты + стаж |
| **Инфраструктура** | 10 | Площадь/ученик + оснащение |
| **Учебная программа** | 10 | Языки + STEM + международные программы |
| **Безопасность** | 5 | Охрана + камеры + медпункт |

### Детализация

Нет лицензии:           0
Базовая лицензия:       3

## Формула расчёта

```python
def calculate_school_rating(school):
    # Объективные метрики
    objective = (
        license_score +           # 0-5
        teacher_score +           # 0-10
        infrastructure_score +    # 0-10
        curriculum_score +        # 0-10
        safety_score              # 0-5
    )  # max 40
    
    # Результаты
    results = (
        graduate_success_score +  # 0-15
        olympiad_score +          # 0-10
        retention_score           # 0-5
    )  # max 30
    
    # Отзывы
    reviews = (
        avg_review_score +        # 0-20
        recommendation_score      # 0-10
    )  # max 30
    
    # Итого
    total = objective + results + reviews  # max 100
    
    return total
```

## Политика прозрачности

### Что публикуем открыто
- Полная методология (этот документ)
- Веса всех критериев
- Как собираем данные
- Как верифицируем отзывы

### Что НЕ влияет на рейтинг
- ❌ Оплата за размещение
- ❌ Реклама на платформе
- ❌ Личные связи
- ❌ Размер школы

### Независимость
- Рейтинговая команда отделена от коммерческой
- Школа не может "купить" позицию
- Любое давление документируется

## Верификация отзывов

```
Методы верификации:
├── Код школы (выдаётся родителям)
├── Подтверждение оплаты (интеграция)
└── Ручная проверка (для спорных случаев)

Отклоняем:
├── Без верификации
├── Подозрение на накрутку (анализ паттернов)
├── Оскорбления и хейт
└── Явно заказные (и + и -)
```

## FAQ

**Q: Почему наш рейтинг упал?**
A: Проверьте changelog. Возможные причины:
- Новые негативные отзывы
- Обновление данных о результатах
- Другие школы улучшились

**Q: Как поднять рейтинг?**
A: Улучшайте реальные показатели:
- Квалификация учителей
- Результаты выпускников
- Качество обучения → хорошие отзывы

**Q: Можно ли оспорить?**
A: Да. Напишите нам с доказательствами неточности данных.
