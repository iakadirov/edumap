# Архитектура маршрутов EduMap.uz

## Общая структура

### 1. Главная страница платформы
**URL:** `/`
**Описание:** Общая информация о платформе EduMap.uz, все типы учебных заведений

### 2. Раздел "Школы"
- **Главная раздела:** `/schools` - лендинг раздела школ
- **Каталог:** `/schools/list` - список всех школ с фильтрами
- **Профиль:** `/schools/[slug]` - профиль конкретной школы

### 3. Раздел "Вузы"
- **Главная раздела:** `/universities` - лендинг раздела вузов
- **Каталог:** `/universities/list` - список всех вузов с фильтрами
- **Профиль:** `/universities/[slug]` - профиль конкретного вуза

### 4. Раздел "Детские сады"
- **Главная раздела:** `/kindergartens` - лендинг раздела детских садов
- **Каталог:** `/kindergartens/list` - список всех детских садов с фильтрами
- **Профиль:** `/kindergartens/[slug]` - профиль конкретного детского сада

### 5. Раздел "Курсы" (учебные центры)
- **Главная раздела:** `/courses` - лендинг раздела курсов
- **Каталог:** `/courses/list` - список всех курсов/центров с фильтрами
- **Профиль:** `/courses/[slug]` - профиль конкретного курса/центра

## SEO-оптимизация

### URL структура
- ✅ Четкая иерархия: `/section/list`, `/section/[slug]`
- ✅ Понятные названия разделов
- ✅ Slug для профилей (SEO-friendly)
- ✅ Нет дублей контента

### Мета-теги
- Уникальные title и description для каждой страницы
- Open Graph теги
- Структурированные данные (JSON-LD)

### Хлебные крошки
- Главная → Раздел → Каталог/Профиль
- Структурированные данные для хлебных крошек

```
│   ├── page.tsx                # Главная раздела вузов (/universities)

│   └── [slug]/
│       └── page.tsx            # Профиль вуза (/universities/[slug])
├── kindergartens/
│   ├── page.tsx                # Главная раздела детских садов (/kindergartens)
│   ├── list/
│   │   └── page.tsx            # Каталог детских садов (/kindergartens/list)
│   └── [slug]/

```

## Адаптация Header

Header должен адаптироваться в зависимости от текущего раздела:
- Активный тип организации в верхней навигации
- Адаптированное меню в нижней строке
- Правильные ссылки на каталог и профили

## Миграция текущих маршрутов

1. `/schools` → `/schools/list` (переместить каталог)
2. Создать новый `/schools` (главная раздела)
3. Обновить все ссылки в коде
