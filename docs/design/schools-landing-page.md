# UI/UX: Главная страница раздела школ — /schools

> **Страница**: `/schools`
> **Тип**: Landing page + точка входа в каталог
> **Версия**: 1.0
> **Дата**: Декабрь 2024

---

## 1. Обзор

### Цель страницы

```
КОНВЕРСИОННЫЕ ЦЕЛИ:

1. Родители → Начать поиск школы
2. Родители → Попробовать AI-подбор
3. Школы → Добавить профиль школы
4. Все → Понять ценность платформы
```

### Целевая аудитория

| Сегмент | Потребность | Что хотят увидеть |
|---------|-------------|-------------------|
| Родители (новички) | Понять что это за сервис | Ценностное предложение, доверие |
| Родители (готовые) | Быстро начать поиск | Поиск, фильтры, каталог |
| Представители школ | Понять выгоду размещения | Преимущества для школ |

**Левая колонка — Проблема:**
| Иконка | Текст |
|--------|-------|
| ❌ | Красивый Instagram ≠ качество обучения |
| ❌ | "Сарафанное радио" субъективно |

| ✅ | Прозрачная связь цены и качества |

#### Иллюстрация 2: Проблема

| Параметр | Значение |
|----------|----------|
| **Описание** | Растерянный родитель среди хаоса информации — телефоны с разными отзывами, вопросительные знаки, противоречивые стрелки, может быть несколько "рекламных" картинок школ которые все выглядят одинаково идеально. |
| **Настроение** | Лёгкий стресс, путаница, информационный шум |
| **Цвета** | Приглушённые, сероватые, может быть лёгкий красный акцент |
| **Размер** | 400×350px |
| **Приоритет** | 🔴 Высокий |

#### Иллюстрация 3: Решение

| Параметр | Значение |
|----------|----------|
| **Описание** | Тот же родитель, но теперь спокойный, с ноутбуком/планшетом, на экране которого структурированный список с рейтингами. Вокруг — порядок, галочки, понятная структура. |
| **Настроение** | Спокойствие, уверенность, ясность |
| **Цвета** | Яркие, чистые, синий/зелёный как акцент |
| **Размер** | 400×350px |
| **Приоритет** | 🔴 Высокий |
