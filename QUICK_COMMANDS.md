# Быстрые команды для работы с проектом

## Переход в директорию проекта

```bash
cd C:\Claude\edumap
```

## Выполнение миграций

После перехода в директорию проекта:

```bash
# Пересоздать базу (если нужно начать заново)
npm run db:reset-cli

# Или просто применить миграции
npm run db:push
```

## Другие полезные команды

```bash
# Запустить dev сервер
npm run dev

# Проверить миграции
npx supabase migration list
```
